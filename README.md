## What is LongClaw?

**LongClaw (龙爪)** is a cross-platform AI desktop assistant that brings top-tier agent capabilities — like those in openclaw and hermes-agent — to regular users who just want to get things done.

- **Ready out of the box** — download, install, and follow the first-run guide. No complex setup.
- **No registration required** — we don't ask for accounts and we don't collect personal data.
- **Powerful agent skills** — read web pages, run commands, organize files, automate browsers, and execute multi-step tasks.
- **Approval gate for risky actions** — deletes, overwrites, command execution, and other high-risk operations pause for your confirmation.
- **ClawHub skill market** — install new skills and MCP servers with one click to extend what LongClaw can do.
- **Multi-channel access** — connect Telegram, Discord, Slack, Feishu, WeChat, and more so your assistant is always within reach.

## Quick Start

1. Download `longclaw-2026.7.3.exe` from the links above.
2. Run the installer and open LongClaw.
3. Follow the first-run guide to add your model API key.
4. Start asking LongClaw to search, write, organize files, or run workflows.

## Highlights

- **Local-first execution** — your data stays on your machine.
- **Transparent updates** — the updater only reports platform, architecture, channel, and version; no file or chat content is uploaded.
- **MCP compatible** — mount standard MCP servers directly from LongClaw.
- **Safety first** — risky operations require explicit user approval.
