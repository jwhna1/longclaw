## 简介

LongClaw 龙爪是一款面向普通用户和小白用户的 AI 桌面助手：

- **开箱即用**：下载安装，无需配置复杂环境
- **免费无门槛**：无需注册，不收集任何隐私
- **强大能力**：查资料、写内容、整理文件、浏览器自动化、连续多步任务执行
- **安全可控**：删除、覆盖、运行命令等高危动作会先弹窗审批
- **技能市场 ClawHub**：一键安装扩展能力，按需增强
