## 龙爪是什么？

**LongClaw（龙爪）** 是一款跨平台 AI 桌面助手，把 openclaw、hermes-agent 级别的智能体能力，做成普通人也能直接上手的工具。

- **开箱即用** —— 下载安装后，跟着首次引导填好模型 API Key 就能用，不用折腾环境。
- **无需注册** —— 不收集任何个人隐私，数据始终在你自己手里。
- **强大的智能体能力** —— 读网页、跑命令、整理文件、浏览器自动化、连续执行多步任务。
- **高危动作先审批** —— 删除、覆盖、运行命令等高风险操作会先弹窗，等你确认后才继续。
- **ClawHub 技能市场** —— 一键安装新技能和 MCP Server，持续扩展龙爪的能力。
- **多渠道接入** —— 支持 Telegram、Discord、Slack、飞书、微信等，让助手随时待命。

## 产品亮点

- **本地优先运行** —— 数据留在本地，不上传文件或聊天内容。
- **透明更新机制** —— 更新器仅上报平台、架构、通道与版本号。
- **兼容 MCP 协议** —— 可直接挂载标准 MCP Server。
